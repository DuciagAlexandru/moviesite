Thank you for visiting this little site.

With this application you use movies genres in order to search for a list of movies based on popularity from IMDB. You input a number to limit the search results and watch a movie trailer for each result. However, before you can start using the search engine you will have to register your account. Then you will have to log in with your credentials after which you can use the Search by genre function located in the upper right corner.

This project was made using the BeautifulSoup plugin in order to scrape some data from IMDB and the Youtube API v3 in order to return movie trailers based on the movies resulted from your genre search. Also this website is powered by Django!

So, let's try it out! :)
